Información Técnica Avanzada

Arquitectura del Modelo

1. Preprocesamiento de Datos

• Normalización de variables predictoras usando StandardScaler

• Imputación de valores faltantes mediante interpolación temporal

• Creación de características lag (1-4 semanas) para capturar autocorrelación

2. Modelos Implementados

Random Forest Regressor

Ensemble de 200 árboles de decisión para capturar relaciones no lineales

Hiperparámetros: max_depth=15, min_samples_split=5

XGBoost

Gradient boosting optimizado para predicciones robustas

learning_rate=0.05, n_estimators=300

SARIMA(2,1,2)(1,1,1)[52]

Modelo de series temporales con componente estacional semanal

AIC: 1245.3, BIC: 1268.7

LSTM (64-32 unidades)

Red neuronal recurrente con 2 capas ocultas

Dropout=0.2, batch_size=16, epochs=100

3. Validación y Optimización

• Validación cruzada temporal (Time Series Split con 5 folds)

• Búsqueda de hiperparámetros mediante Grid Search con 100 iteraciones

• Ensemble por stacking: meta-modelo Ridge Regression

• Validación en conjunto de test: últimas 12 semanas (hold-out)

4. Sistema de Alertas y Umbrales

• Umbral Riesgo Bajo: < 18 µg/L (basado en percentil 33)

• Umbral Riesgo Medio: 18-25 µg/L (percentil 33-66)

• Umbral Riesgo Alto: > 25 µg/L (percentil 66+, activación de alerta)

• Sistema de notificaciones: email + SMS para autoridades ambientales
